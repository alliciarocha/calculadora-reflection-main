# 🧮 Calculadora Utilizando Reflection 
Este projeto implementa uma calculadora simples utilizando Reflection em Java, permitindo invocar métodos dinamicamente para realizar operações aritméticas.
## ⚙️ Operações
- ➕ Soma
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão

## 🛠️ Instalação

```bash
mvn clean install
```

## 🚀 Execução
````bash
mvn exec:java
````
